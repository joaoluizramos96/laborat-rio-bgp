# Laboratório BGP - EVE-NG (Cisco)

Utilizadas as imagens Cisco L3-ADVENTERPRISEK9-M-15.5-2T.bin e Virtual PC do próprio emulador.

Realizado, testado e validado na versão EVE-NG 6.2.0-3-Community
